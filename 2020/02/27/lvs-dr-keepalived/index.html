一、Keepalived

主要提供负载均衡和高可用功能，负载聚恒实现依赖于 Linux 的虚拟服务内核模块(ipvs),而高可用是通过 VRRP 协议实现多台机器之间的故障转移服务

VRRP(Virtual Router Redundancy Protocol 虚拟路由协议),VRRP (Virtual Router Redundancy Protocol-虚拟路由冗余协议)，首先我们要知道VRRP是一种容错性协议，它是通过将多台设备虚拟化成一台设备，如果其中一台设备出现故障，那么另一台设备可以迅速接替其工作，已保证通讯的可靠性和连续性。
